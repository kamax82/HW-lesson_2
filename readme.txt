Папка 'home_work' соержит файлы:
 - conftest.py  с фикстурами для четырех тестируемых типов данных. Фикстура генирирует объект опредеенного типа, для
 тестирования встроеных методы каждой из структур данных в Python.
 - test_dictionary файл содержаший тестовые функции для структур данных - Словарь. 5я функция параметризованная,
 принимает дополнительные параметры для теста. Выполняется трижды.
 - test_list файл содержаший тестовые функции для структур данных - Список. 5я функция параметризованная.
 Выполняется трижды, каждый раз принимая новое значение.
 - test_set файл содержаший тестовые функции для структур данных - Множество. 5я функция параметризованная. Выполняется
 четыре раза, принимая новое значение на каждой итерации. тем самым увеличивая длинну (размер) множества.
 - test_string файл содержаший тестовые функции для структур данных - Строка. 5я функция параметризованная. выполняется
 один раз, в качестве дополнительного параметра приниает подстроку, участвующую в проверке метода замены подстроки в стрке.
